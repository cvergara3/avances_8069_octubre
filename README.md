# Avances Octubre Proyecto 8069
El presente repositorio incluye los códigos programados para recontruir y submuestrear shot gathers sintéticos y reales con diferentes tipos de submuestreo: capa binaria entrenable con STE, random, uniform y jittered. Consta de dos códigos:

1. submuestreo_tipos_sinteticos.ipynb: Código de reocnstrucción U-Net con diferentes tipos de submuestreo en datos sintéticos de la cuenca Cordillera Oriental
2. submuestreo_tipos_reales.ipynb: Código de reocnstrucción U-Net con diferentes tipos de submuestreo en datos reales de Utah Forge
